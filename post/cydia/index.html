插件列表

1. 电话助手 （付费软件20元）

电话助手完美搭配原生UI和加强原生App功能 拥有来电窗口,电话录音,T9拨号键盘,来去电归属地,骚扰拦截,黑名单拦截等强势功能，状态栏时间网速显示 啥啥都有

2. QQ净化

可以关闭qq上一些乱七八糟、花里胡哨的东西，还可开启闪图和消息防撤回等等

3. 微信净化

可以开启微信防撤回、开启微信Callkit、还可以开启朋友圈30秒视频等等

4. shadow屏蔽越狱检测

一款用来屏蔽越狱检测的插件

5. Titan视频画中画

可以在iOS13上开启画中画功能，自带的Safari浏览器、b站、vvebo等都支持画中画的功能了

6. Flex 3

Flex是一款强大的越狱插件，我主要用来屏蔽一些广告
分享一个VIP账号
sundasheng521@qq.com 密码7758521a

7. Filza File 文件管理

比苹果自带的文件管理器不知道高到哪里去了

8. APP Store++ 应用降级

可以把你想降级的软件降级到任何版本，而且还可以屏蔽升级提示
